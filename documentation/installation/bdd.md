## Configuration de la base de données


* Il faut tout d'abord configurer le .env pour la connection à la base 

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=woozbest
DB_USERNAME=root
DB_PASSWORD=
```

* Il faut ensuite importer les scripts sql (fichier .sql du répertoire sql)
